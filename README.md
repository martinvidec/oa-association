# OpenAustria (OA) – Association

Vorlagen und Arbeitsunterlagen zur Gründung des gemeinnützigen Vereins
**„OpenAustria (OA) – Verein zur Förderung KI-gestützter Open-Source-Software"**
mit Sitz in Wien, Österreich.

## Zweck des Vereins

Förderung von Open-Source-Software in Österreich, insbesondere KI-gestützter Open-Source-Anwendungen. Der Verein verfolgt ausschließlich und unmittelbar gemeinnützige Zwecke im Sinne der §§ 34 ff BAO (Volksbildung, Wissenschaft, Forschung, digitale Teilhabe).

## Struktur des Repositories

```
.
├── docs/
│   ├── requirements/               # Rechercheunterlagen zur Vereinsgründung
│   │   └── vereinsgruendung-oesterreich.md
│   └── documents/                  # Gründungsdokumente (Vorlagen mit Platzhaltern)
│       ├── 01-statuten.md
│       ├── 02-gruendungsprotokoll.md
│       ├── 03-anzeige-vereinserrichtung.md
│       ├── 04-mitgliedsantrag.md
│       ├── 05-datenschutzerklaerung.md
│       └── README.md
├── PRIVATE/                        # lokal, nicht eingecheckt (siehe .gitignore)
├── LICENSE
└── README.md
```

## Nutzung

1. `docs/requirements/` enthält die rechtlichen Grundlagen (VerG 2002, BAO-Gemeinnützigkeit, Gebühren, Ablauf).
2. `docs/documents/` enthält die Vorlagen mit Platzhaltern `[...]`.
3. Die ausgefüllten Versionen mit Realdaten (Namen, Adressen, Geburtsdaten etc.) gehören in den Ordner `PRIVATE/`, der von Git ignoriert wird.

## Mitmachen

Interessiert an Mitarbeit? Issues und Pull Requests sind willkommen — insbesondere:
- Verbesserungen der Statutenformulierung
- Ergänzungen zur Gemeinnützigkeit / Finanzamt-Praxis
- Übersetzungen (EN)
- Checklisten / Hilfsskripte
